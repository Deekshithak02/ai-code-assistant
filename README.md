# 🤖 AI Code Assistant

An AI-powered Code Assistant that analyzes Java code and provides:

- 🔍 Bug Detection
- 🔒 Security Issue Detection
- 💡 Code Improvement Suggestions
- ⚡ Time Complexity Analysis

---

## 🚀 Features

- Analyze Java source code
- Detect bugs automatically
- Identify security issues
- Suggest code improvements
- Estimate time complexity

---

## 🛠️ Tech Stack

- Python
- LangChain
- Google Gemini API
- JSON
- Git
- GitHub

---

## 📁 Project Structure

AI-Code-Assistant/
│── main.py
│── retrieval.py
│── embedding_store.py
│── requirements.txt
│── embeddings.json
│── sample.java

---

## ▶️ Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Add your Gemini API Key inside the `.env` file.

4. Run the project

python main.py
